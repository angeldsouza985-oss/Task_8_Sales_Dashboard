# Task 8 - Simple Sales Dashboard Design

## Objective
Create an interactive sales dashboard to visualize sales performance.

## Tools Used
- Power BI Desktop
- Excel Dataset

## Dashboard Components
1. Sales over Months (Line Chart)
2. Sales by Country (Bar Chart)
3. Sales by Product (Donut Chart)
4. Country Slicer

## Key Insights
- Sales peaked in January and June 2022.
- Australia and the UK are strong contributors.
- Product revenue is distributed across multiple products.
- Users can filter results using the country slicer.
